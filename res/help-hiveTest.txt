--------- Help ----------------
CC : current control (in red)
MW : mouse wheel
RM: mouse right button
CP : cursor position
-------------------------------
Space	: toggle pause
D	: toggle debug mode
MW	: Zoom
Esc	: close window
C	: reload config. file
Arrows	: move simulation view
Tab	: switch to next control
R	: regenerate simulation
         (random world built
	 from json)
W	  : toggle show humidity
F         : add Flower at CP
L	  : load world from file
O	  : save world to file
H	  : add hive at CP
B	  : add bee to hive
